# LSTC---Lenovo-Service-Tool-Center
LSTC (Lenovo Service Tool Center) utility is a windows-based service support platform utility to all field service technician.
Why make it? :
There is no universal tool center
platform for lenovo service team.
Service engineer need download different
software for servicing different product lines.
Meanwhile, now a days, more and more
product line start to enable some unique
service utility, that is big challenge to
service delivery.
So, we made it.

The Benefits :
Field technician can easily get all
needed service support resource via
LSTC (Lenovo Service Tool Center)
utility .
- Simplified FS SOP.
- Shorten ART
- Improve CX
LSTC ‘s log feature could help show the usage
volume and help on field performance
monitoring.
LSTC ‘s user ID management feature(TBD) will
help on service delivery technical management
